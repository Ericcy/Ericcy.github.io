版本控制的介绍

版本控制工具 VCS

Version Control System

将代码每次更改之后的结果进行保存，保存为指定的版本。

VCS可以实现的效果就是，给每一次修改都保存一次历史记录。

版本控制工具分类

本地式版本控制工具

版本库存放在本地，每个人都有自己的版本库，跟别人毫无关系，在单人开发当中，本地版本控制系统可以用来管理代码，但是如果是多人开发，就没办法使用了

集中式版本控制工具

将版本库集中管理，每次修改的内容都需要在这个集中管理的版本库中生成历史记录！ （SVN）
